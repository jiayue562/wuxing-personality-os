# 五行人格心理学OS (Wuxing Personality OS)

> **镜 · 灯 · 刀** —— 凤爪OS是"镜"，凤心OS是"灯"，凤脑OS是"刀"

## 概述

五行人格心理学OS是一套基于中国五行哲学（木、火、土、金、水）的**人格心理学智能体操作系统**。它由三层架构组成，模拟人类从感知到认知到行动的完整心智流程：

| 层级 | 组件 | 定位 | 类比 |
|:----|:----|:----|:----|
| L1 | **凤爪OS** | 知觉接待台 | "镜" —— 纯粹感知 |
| L2 | **凤心OS** | 智能体发动机 | "灯" —— 场景路由 |
| L3 | **凤脑OS** | 知识地基层 | "刀" —— 理论支撑 |

## 核心架构

### 凤爪OS — 知觉接待台
庄子"三无"心法为感知基底：无己则镜、无功则空、无名则照。
- ① 情绪温度、② 意图方向、③ 场景识别、④ 能量状态（四维感知）
- 15类标准场景映射（S01-S15）
- 与凤心OS的JSON协作协议

### 凤心OS — 智能体发动机
1+5模式：1个快思考总调度 + 5个慢思考分智能体
- 自动触发机制：感知上下文→识别场景→自动调度引擎
- 无需关键词匹配，基于场景理解
- 三层感知雷达整合

### 凤脑OS — 知识地基层
17篇L7理论基石（80,000字）+ 130+条跨域联系网络
- 东西方认知科学融合
- 为凤心OS提供深度理论支撑
