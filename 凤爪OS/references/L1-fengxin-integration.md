# L1-凤心集成接口 (L1-Fengxin Integration)

> 凤爪OS与凤心OS的标准化对接协议，确保数据流在两系统间无损传递。

## 一、接口定位
凤爪OS是五行人格心理学OS的L1知觉接待台，凤心OS是L2智能体发动机。
凤爪OS完成感知后，将结构化的感知数据传递给凤心OS进行场景路由和智能分析。

## 二、数据传递格式

### 输入数据（凤爪OS → 凤心OS）
| 字段 | 类型 | 说明 | 示例 |
|------|------|------|------|
| raw_text | str | 用户原始文本 | "最近和伴侣总是吵架" |
| emotion_temp | int(0-10) | 情感温度评分 | 3 |
| energy_state | str | 能量状态 | "low/exhausted" |
| scene_labels | list[str] | 场景分类标签 | ["intimate", "conflict"] |
| intent | str | 用户意图 | "seeking_advice" |
| confidence | float(0-1) | 感知置信度 | 0.85 |

### 输出数据（凤心OS → 凤爪OS）
| 字段 | 类型 | 说明 |
|------|------|------|
| result_type | str | 结果类型（direct/diagnosis/report） |
| content | str | 处理结果正文 |
| wuxing_analysis | dict | 五行分析报告（可选） |
| action_suggestions | list | 行动建议列表（可选） |

## 三、调用时序
1. 凤爪OS接收用户输入 → 感知意图/情感/场景/能量
2. 凤爪OS附加情感温度 → 调用凤心OS路由接口
3. 凤心OS快思考 → 五行归类 → 路由分智能体
4. 分智能体处理 → 结果返回凤心OS
5. 凤心OS整合 → 返回凤爪OS
6. 凤爪OS格式化 → 输出给用户

## 四、异常处理
| 异常场景 | 处理策略 |
|---------|---------|
| 凤心OS不可用 | 凤爪OS独立处理，降级为通用对话 |
| 场景识别冲突 | 以凤爪OS感知为准，凤心OS二次确认 |
| 超时（30s无响应） | 自动降级，返回凤爪OS初步回应 |
| 数据格式错误 | 日志记录，尝试自动修复 |
