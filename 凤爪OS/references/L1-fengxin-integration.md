# L1-凤心集成接口

> 凤爪OS与凤心OS的标准化对接协议

## 数据传递格式
- 输入：用户原始文本 + 情感温度评分（0-10）+ 场景分类标签
- 输出：凤心OS处理结果 + 五行分析报告（可选）

## 调用时序
1. 凤爪OS接收用户输入 → 感知意图
2. 凤爪OS附加情感温度 → 调用凤心OS路由接口
3. 凤心OS快思考 → 五行归类 → 路由分智能体
4. 结果原路返回 → 凤爪OS格式化输出

## 异常处理
- 凤心OS不可用 → 凤爪OS独立处理，降级为通用对话
- 场景识别冲突 → 以凤爪OS感知为准，凤心OS二次确认
- 超时机制 → 30s无响应自动降级

## 版本兼容
- 凤爪OS v2.0+ 兼容凤心OS v4.0+
- 接口变更需双向通知
