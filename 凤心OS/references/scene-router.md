# 场景路由模块 (Scene Router)

> 凤心OS的核心路由引擎，根据用户输入自动映射到对应的五行场景

## 路由流程
1. 接收凤爪OS传递的场景分类结果
2. 匹配五行类型（木/火/土/金/水）
3. 路由到对应分智能体
4. 返回结果整合

## 场景-五行映射矩阵

| 场景 | 主五行 | 次五行 | 路由目标 |
|------|--------|--------|---------|
| 亲密关系 | 水 | 火 | 水智能体+火智能体 |
| 亲子 | 土 | 木 | 土智能体+木智能体 |
| 领导力 | 火 | 金 | 火智能体+金智能体 |
| 团队 | 金 | 土 | 金智能体+土智能体 |
| 沟通 | 水 | 火 | 水智能体+火智能体 |
| 健康 | 土 | 水 | 土智能体+水智能体 |
| 生活 | 木 | 火 | 木智能体+火智能体 |
| 测评 | 全部 | — | 五行全开 |

## 路由优先级
- 显式声明 → 隐式推断 → 默认路由
- 多场景交织时按权重排序，取Top2

## 错误处理
- 未识别场景 → 降级到通用对话模式
- 路由超时 → 重试1次后启用备用路径
